RequestAbilityPacket (id: 184)

Purpose: Sent from client to server. Used to request an ability change.
Notes: Once changed, the server will broadcast the updated state of abilities for that player. If the request is rejected, the caller will receive their reverted state of Abilities. Can only be used to modify the calling player. - mVariable: Info about this variable
